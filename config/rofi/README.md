## Rofi com BLUR

- **Dependências**:
  - `rofi`
  - `compton-tryone-git`
  
#### Para executar conforme a config padrão:

`rofi -modi drun -show drun`

#### Modo dashboard

`rofi -show drun -theme ~/.config/rofi/appmenu/drun.rasi`

<br></br>

`rofi -show drun -theme ~/.config/rofi/sidebar/rofi.rasi`
